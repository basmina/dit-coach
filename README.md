# ⚡ DoIt Coach — AI Accountability Agent

> *You know what you should do. DoIt makes sure you actually do it.*

DoIt is a conversational AI accountability coach that helps you stay consistent with your goals. Unlike passive apps and reminders, DoIt talks back — it pushes you, challenges your excuses, and celebrates your wins.

---

## 🎯 The Problem

Most people don't fail because they lack knowledge.
They fail because they lack accountability.

To-do apps are passive. Reminders are easy to ignore. What actually works is having someone who asks *"did you do it today?"* and doesn't accept weak excuses.

That's DoIt.

---

## ✨ Features

- 💬 **Conversational coaching** — talk to your coach naturally, like texting a friend
- 🎯 **Goal tracking** — set up to 3 goals and the coach always brings them up
- 🔥 **Excuse challenger** — the coach gently pushes back when you're slacking
- 🏆 **Win celebration** — genuine encouragement when you follow through
- 🧠 **Context memory** — remembers what you said earlier in the conversation
- 📱 **Clean UI** — works on desktop and mobile

---

## 🛠️ Built With

- **Node.js** — backend server
- **Express.js** — API routing
- **Claude API (Anthropic)** — AI coach brain
- **HTML/CSS/JS** — frontend chat interface
- **Render** — deployment

---

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Anthropic API key → [console.anthropic.com](https://console.anthropic.com)

### Installation

```bash
# Clone the repo
git clone https://github.com/basmina/dit-coach.git
cd dit-coach

# Install dependencies
npm install

# Create .env file
echo "ANTHROPIC_API_KEY=your-key-here" > .env

# Run the app
node server.js
```

Then open **http://localhost:3000** in your browser.

## 💡 How to Use

1. **Add a goal** — type something you've been putting off and click "+ Add Goal"
2. **Start talking** — tell the coach how it's going
3. **Be honest** — the coach works best when you're real with it
4. **Check in daily** — consistency is the whole point
